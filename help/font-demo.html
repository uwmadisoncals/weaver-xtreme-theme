Web Fonts - Not displayed on Android or iOS devices!

The following Web fonts are supported by most modern desktop browsers. Ideally, you should see examples of the named font. If your browser doesn't support a specific Web font, it will use a fallback font. Many mobile browsers, especially Android based browsers, will not show many of these examples (iOS will display many Web Fonts, but not all), and you'll see many of them displayed in the browser's default serif, sans-serif, mono, or cursive font. If you use the Google Fonts supported by Weaver Xtreme or Weaver Xtreme Plus, or add other fonts using the advanced font support of Weaver Xtreme Plus, for your site, they will be correctly displayed on essentially all modern browsers. Since Android browsers still do not support the standard Web Safe fonts, we highly recommend that you use one of Google fonts included with Weaver Xtreme or added fonts from Weaver Xtreme Plus!
